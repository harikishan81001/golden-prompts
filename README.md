## 🧠 What This Repository Provides

### 1. Code Review Prompt (`code-review-python.md`)

A **comprehensive review framework** that evaluates:

* Code quality and design
* Architecture and scalability
* Business logic correctness
* Docker & infrastructure setup
* Security vulnerabilities
* Test coverage and edge cases
* Performance bottlenecks

👉 Designed for **Staff+ level reviews**, not superficial linting.

---

### 2. Documentation Prompt (`documentation.md`)

A **production-grade documentation generator** that ensures:

* Clear architecture explanation
* Execution and data flow visibility
* Accurate diagrams (Mermaid / draw.io)
* Operator-friendly documentation
* No speculative or irrelevant content

👉 Useful for:

* Onboarding engineers
* Design reviews
* System understanding & debugging

## 🛠️ How to Use

1. Pick the relevant prompt:

   * Code Review → `code-review-python.md`
   * Documentation → `documentation.md`

2. Copy the prompt into your preferred LLM (Claude, ChatGPT, etc.)

3. Provide:

   * Codebase / PR diff (for review)
   * System details (for documentation)

4. Review output and iterate if needed
